Easy schema definition

Define your schemas and choose from multiple form widgets and validation options. 


const price = buildProperty({
  name: "Price",
  dataType: "number",
  validation: {
    required: true,
    min: 0,
    max: 1000
  }
});

Built for every project

Use integrated hooks and callbacks to integrate your business logic in multiple ways. 


const productCallbacks = buildEntityCallbacks({
  onPreSave: ({ values }) => {
    // Example: Ensure name is uppercase
    if (values.name) {
      values.uppercase_name = values.name.toUpperCase();
    }
    return values;
  }
});
